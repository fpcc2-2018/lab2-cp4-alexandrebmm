1. Qual a nossa taxa de cliques diria geral? Como ela varia entre os grupos?

A taxa de cliques" a proporo de sesses de busca em que o usurio clicou em um dos resultados apresentados.

A partir de ento, identificamos a varivel num_clicks como a quantidade de vezes que um usurio visita uma pgina a partir dos resultados de uma busca (valor visitPage para a varivel action do dataset original).

set.seed(3)

buscasCliques = buscas %>%
  filter(results >= 1, !is.na(num_clicks))

fun.boot = function(dado, indices) {
  
  taxa_cliques <- dado %>%
    slice(indices) %>%
    arrange(date) %>%
    group_by(date, num_clicks) %>%
    summarise(n = n()) %>% 
    mutate(taxa_cliques = n / sum(n) * 100) %>%
    ungroup()

  
  taxas = taxa_cliques %>%
    filter(num_clicks > 0) %>%
    group_by(date) %>%
    summarise(taxa_cliques = sum(taxa_cliques)) %>%
    pull(taxa_cliques)
  
  return(mean(taxas))
}

res.boot <- boot(data = buscasCliques, 
                 statistic = fun.boot, 
                 R = 2000)

4. A durao da sesso aproximadamente o tempo entre o primeiro e o ltimo evento de uma sesso. Escolha uma das variveis presentes dos dados e descreva sua relao com o tamanho da sesso. Visualize o relacionamento.

A durao da sesso ser sumarizada na varivel session_length, a partir da diferena entre o final de uma sesso e o seu incio, utilizando para tal a grandeza de tempo segundos, uma medida discreta.

Escolhemos a varivel group para relacionar com a durao de sesso, tendo como objetivo identificar se os usurios de um dos grupos tende a permanecer mais tempo online.

Para garantir consistncia nos dados, filtraremos apenas as sesses com durao maior que 0 segundos, visto que uma sesso de busca com tempo zero (ou negativo) uma anomalia.

pergunta4 = buscas %>%
    group_by(session_id) %>%
    mutate(
        session_length = as.numeric(
            difftime(last(session_end_date), first(session_start_date),
                     tz = "UTC",
                     units="secs"))) %>%
    filter(session_length > 0)

pergunta4 %>% 
    ggplot(aes(x = group, y = session_length, col = group)) + 
    geom_boxplot() +
    scale_y_log10() +
    labs(title="Relao entre grupos e durao de sesso",
         subtitle="Grupos vs. Durao de sesso",
         x="Grupos",
         y="Durao de sesso (em segundos)",
         caption = "Fonte: Wikimedia Foundation",
         fill="Grupos")

Por fim, observamos atravs do grfico apresentado acima que o Grupo A tende a ter sesses de busca maiores do que o Grupo B, inclusive com pouco mais de 75% do dados de A sendo maiores que 50% dos dados de B.
